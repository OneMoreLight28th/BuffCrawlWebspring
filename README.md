# BuffCrawlWebspring

## 项目概述

该项目是一个基于Spring Boot和MongoDB的Web应用程序，用于提供网易buff中虚拟物品的交易信息查询服务。用户可以通过关键字搜索物品，查看物品的详细信息和交易历史，可以用来练习java的相关技术栈。

## 技术栈

- 后端：Spring Boot 
- 数据库：MongoDB + MySQL
- 前端：Vue.js
